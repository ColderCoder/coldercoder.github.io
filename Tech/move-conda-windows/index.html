Windows 迁移Anaconda到其他分区

ColderCoder
2020-10-14
Tech
trick

  1. 编辑环境变量并移动

    此电脑 -> 属性 -> 高级系统设置 -> 高级 -> 环境变量 或者Win+S搜索”PATH”进入“编辑系统环境变量”。

    根据安装时选择的为所有用户安装/为当前用户安装，分别查看系统PATH/用户PATH, 找到对应路径，移动到想要的目录，并修改相关PATH条目为新的位置即可。

    echo %PATH% 或者 重启计算机 使对PATH的更改生效。

    执行conda -V检查是否迁移成功。

  2. 重装pip

    重装pip以解决pip Fatal error in launcher 错误:

    python -m pip uninstall pip 卸载pip,
    cd 到 Anaconda3\Scripts 目录下，
    执行 easy_install.exe pip 以安装pip.

  3. 后续

    一怒之下卸载了，安装在WSL2里。那样也方便环境迁移，带上一个VHDX文件就走，也防止系统崩溃不得不格式化C盘的狼狈。
    白忙活（笑）
